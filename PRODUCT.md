# Продукт: nikamet.pro (metal-seo)

## Суть работы

SEO-сайт по металлопрокату (Astro + React): справочные страницы по ГОСТам с калькуляторами веса для труб, листов, швеллеров, двутавров, углов, шпунта. Данные из CSV (ручная база ГОСТов) → JSON → статические страницы. Цель — трафик по запросам «вес трубы X», «лист стальной», «сколько в тонне» и т.д., лид-форма для заявок.

## Плюсы

- Единый источник формул и данных: CSV → import-csv → JSON, типизация в `lib/types.ts`.
- Раздельные калькуляторы под тип продукта: трубы (с погрузкой в шаланду), листы (поштучно), профиль (по длине).
- Расширяемая база ГОСТов через CSV в `gosts_data_manual/`.
- SSR/статическая генерация, быстрая выдача.

## Минусы

- Формулы дублировались в компонентах вместо одного модуля `lib/calculator.ts`.
- Риск расхождения расчёта с ГОСТ/эталоном (23met.ru), если коэффициент или единицы измерения отличаются.
- Нет явной поддержки нержавейки/цветмета (разная плотность).
- Калькулятор на `/calculator/` общий, без привязки к конкретному ГОСТу/размеру.

## УТП (уникальное торговое предложение)

- Калькулятор веса + погрузка в шаланду (трубы) — наглядная укладка и количество машин.
- Страницы под каждый размер по ГОСТу (труба 57×3.5, лист 10×1500×6000 и т.д.) с корректной массой п.м. / листа из справочника.
- Прозрачные формулы (ГОСТ): вес трубы по 0,02466·S·(D−S), лист по толщине и размерам в мм.

## Рекомендации по развитию

1. **Калькулятор**: сделан единый источник формул в `lib/calculator.ts` (труба по ГОСТ 0,02466×S×(D−S)×L), дубликаты в компонентах убраны.
2. **Расширение базы ГОСТов**: инструкция и форматы CSV — в `gosts_data_manual/README.md`. Можно добавить ГОСТ 8639-82 (квадратные трубы), 3262-75 (ВГП) по тому же принципу.
3. **Нержавейка/цветмет**: опциональный коэффициент плотности в калькуляторе (по умолчанию 7850).
