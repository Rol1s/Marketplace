01 / Суть проекта

Что это и как работает

Принцип pSEO

Один шаблон + база данных = тысячи уникальных страниц. Каждая под конкретный long-tail запрос: "труба 219×8 ст20 купить Новосибирск"

Монетизация

Каждая страница: таблица ГОСТа + калькулятор веса + цена → форма заявки (LeadForm). Пользователь ищет → находит точный размер → оставляет заявку → лид идёт поставщику.

Стек

Astro SSG → статический HTML, быстрый, индексируемый. React → интерактивные калькуляторы. TypeScript → типизация данных. JSON → база ГОСТов. Деплой → Vercel.
